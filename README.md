# BNV User Management System

This is a full-stack MERN application developed as part of a technical assessment. 

### Features:
- **Responsive UI:** Built with React-Bootstrap.
- **Backend API:** Node.js/Express with MongoDB.
- **Functions:** Full CRUD, Search, Pagination, and Export to CSV.

### Setup:
1. Clone the repo.
2. Run `npm install` in both `/Frontend` and `/Backend`.
3. Add your `.env` to the Backend folder.
4. Run `node index.js` (Backend) and `npm start` (Frontend).
